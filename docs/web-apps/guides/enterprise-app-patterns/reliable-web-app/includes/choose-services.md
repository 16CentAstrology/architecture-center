## Choose Azure services

To minimize the replatforming effort, choose services that support the features of your web app, such as the same runtime and database engine. Prefer managed services to improve security and reduce management overhead.

Use the following table to find guidance to choose the right service for each web app component. It lists a component recommendation, the reference implementation selection, and a link to more decision guidance. If you're unsure which service to choose, start with the reference implementation.
